## 你是否知道自己要做什么？

公司很忙，业务很忙。自己怎么办？  很多年过去了你会留下些什么？

> 一定要定期给自己做一个复盘【深刻认识到】

- 项目中的难点、亮点一定要记录
- 使用到的知识点、技能点可以琐碎的记录

> 现在意识到晚吗？ 不晚，只要马上开始做起来就不晚！

## 如何提升效率

每做一件事情的时候，都要花一分钟思考下，该任务可以分成多少个模块，然后模块间如何被统筹.

如何做好规划，整理思路，避免把整块时间变得支离破碎。
